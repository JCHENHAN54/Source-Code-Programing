编程世界

编程的世界是建立在严密的逻辑之上，不存在任何的随机性。对于追求确定性的人来说，
这个世界让人极度舒适。编程的造物空间更庞大，理论上来说，你可以搭建一个无限宽
广的『世界』，实现无穷的功能，还不需要什么成本。很多人自学编程的起步都是开发
游戏，因为游戏的造物感更具象。反过来说，如果不喜欢创造，几乎不可能喜欢编程，
也不可能写出好的程序。

The world of programming is based on strict logic, and there is no randomness. For those who
pursue certainty, this world is extremely comfortable. The creation space of programming is
even larger. In theory, you can build an infinitely wide "world" and realize infinite functions
without any cost. Many people start to develop games for self-learning programming, because

games have a more visual sense of creation. On the other hand, if you don't like creativity, it is
almost impossible to like programming, and it is impossible to write good programs.
